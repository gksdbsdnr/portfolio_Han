# 자기소개서 포트폴리오

취업 지원용 자기소개서와 포트폴리오를 GitHub Pages에 배포하기 위한 정적 HTML 사이트입니다.
ㅇ
## 수정할 곳

- `index.html`: 이름, 직무, 경력, 프로젝트, 연구, 학력, 연락처 문구 수정
- `script.js`: 프로젝트 상세 팝업 내용 수정
- `styles.css`: 색상, 간격, 반응형 디자인 수정
- `assets/profile-visual.svg`: 프로필 비주얼 교체

## GitHub Pages 배포

1. GitHub에 새 저장소를 만듭니다.
2. 이 폴더의 파일을 저장소 루트에 업로드합니다.
3. 저장소의 `Settings > Pages`로 이동합니다.
4. `Build and deployment`에서 `Deploy from a branch`를 선택합니다.
5. Branch를 `main`, 폴더를 `/root`로 선택한 뒤 저장합니다.

배포 후 `https://사용자명.github.io/저장소명/` 주소로 확인할 수 있습니다.
